Usa este sistema de colores para la app:

MARCA
- Primary:       #0EA5E9  → botones, links, nav activo
- Primary dark:  #0284C7  → hover y pressed
- Primary light: #E0F2FE  → fondos de badges y chips
- Accent:        #22D3EE  → highlights, tabs seleccionados

SEMÁNTICOS
- Success:  #10B981  → confirmado, pagado, completado
- Warning:  #F59E0B  → en espera, en proceso
- Error:    #EF4444  → cancelado, error, fallo
- Info:     #8B5CF6  → notificaciones, tips

NEUTROS
- Text primary:   #0F172A
- Text secondary: #64748B
- Border:         #E2E8F0
- Surface:        #F1F5F9  (cards, inputs)
- Background:     #F8FAFC  (fondo general)
- White:          #FFFFFF  (modales, sheets)

REGLAS
- Nunca usar Primary en fondos grandes, solo en elementos interactivos
- Success/Warning/Error solo para estados, nunca decorativo
- Accent solo como highlight, no como botón principal
